# 🔍 TechDetector

**A Wappalyzer-like Web Technology Fingerprinting Tool** with version detection and basic vulnerability checking.

## ✨ Features
- Detects technologies (CMS, Frameworks, Servers, CDNs, Libraries)
- Extracts version numbers when possible
- Highlights potentially vulnerable versions
- CLI Tool + Streamlit Web Interface
- JSON Report Export
- Beautiful terminal output

## 🚀 Quick Start

```bash
git clone https://github.com/Hacha0001/TechDetector.git
cd TechDetector
pip install -r requirements.txt
